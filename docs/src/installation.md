# Installation

## Cargo (Recommended)

```bash
cargo install neatcli
```

## Homebrew (macOS/Linux)

```bash
brew tap patchybean/tap
brew install neatcli
```

## From Source

```bash
git clone https://github.com/patchybean/neatcli.git
cd neatcli
cargo build --release
```

## Verify Installation

```bash
neatcli --version
```
