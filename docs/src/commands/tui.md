# tui

Launch interactive TUI (Text User Interface) mode.

## Usage

```bash
neatcli tui [PATH]
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑/↓` | Navigate files |
| `Space` | Select/deselect file |
| `a` | Select all |
| `1-7` | Choose organize mode |
| `Enter` | Preview moves |
| `e` | Execute moves |
| `q` | Quit |

## Examples

```bash
# Open TUI in Downloads
neatcli tui ~/Downloads

# Open TUI in current directory
neatcli tui
```
