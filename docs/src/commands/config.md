# config

Manage configuration files.

## Usage

```bash
neatcli config <ACTION>
```

## Actions

| Action | Description |
|--------|-------------|
| `init` | Create sample config file |
| `show` | Display current config |

## Examples

```bash
# Create config file
neatcli config init

# Show current config
neatcli config show
```

## Config Location

Default: `~/.neat/config.toml`
